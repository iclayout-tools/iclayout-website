# IC Layout Website

Modern, professional website for IC Layout - Intelligent Concepts and Layouts.

## About IC Layout

From precision IC layout design to transformative software solutions. We specialize in:
- Integrated Circuit Layout Design
- Custom Software Development
- Business Efficiency Solutions

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Project Structure

```
├── app/
│   ├── layout.tsx       # Root layout
│   ├── page.tsx         # Home page
│   └── globals.css      # Global styles
├── public/              # Static assets
└── ...config files
```

## Deployment

This site is configured for seamless deployment on Vercel:

1. Push to GitHub
2. Connect repository to Vercel
3. Vercel automatically builds and deploys
